{% trans "A timeline of Mozilla’s work on this issue" %}

  1. {% trans "July 2019" %}

    {% trans "Mozilla awards Guillaume Chaslot, a former YouTube engineer, a fellowship to support his work investigating AI systems, including the impacts of YouTube’s recommendation engine." %}

  2. {% trans "August 7, 2019" %}

    {% blocktrans with link="https://foundation.mozilla.org/blog/youtubes-recommendations/" trimmed %} Mozilla announced our intentions to investigate YouTube’s recommendation problem, and to begin a dialogue with the company about making improvements. {% endblocktrans %}

  3. {% trans "August 14, 2019" %}

    {% blocktrans with link="https://foundation.mozilla.org/blog/what-can-you-trust-internet/" trimmed %} Mozilla published a reading list for our community detailing the ways recommendation AI — including YouTube’s — can manipulate consumers. {% endblocktrans %}

  4. {% trans "September 3, 2019" %}

    {% blocktrans with link="https://foundation.mozilla.org/blog/answers-your-questions-about-dark-side-internet/" trimmed %} Mozilla hosted a Q&A with a teenager who was radicalized online, partially due to YouTube’s recommendation AI. {% endblocktrans %}

  5. {% trans "September 10, 2019" %}

    {% blocktrans with link="https://foundation.mozilla.org/blog/youtube-regrets/" trimmed %} Mozilla launched its #YouTubeRegrets campaign, urging users to share stories about harmful recommendations they encounter. {% endblocktrans %}

  6. {% trans "October 14, 2019" %}

    {% blocktrans with link="https://foundation.mozilla.org/blog/our-recommendation-youtube/" trimmed %} After meeting with YouTube staff to discuss this issue, Mozilla published three concrete actions the platform could take to help address its recommendation problem. These changes entail making YouTube’s recommendation data more transparent. {% endblocktrans %}

  7. {% trans "October 15, 2019" %}

    {% blocktrans with link="https://foundation.mozilla.org/blog/28-reasons-why-youtube-must-change/" trimmed %} Mozilla published findings from our #YouTubeRegrets research, highlighting how users were recommended anti-science content, anti-LGBT content, and other harmful videos. {% endblocktrans %}

  8. {% trans "October 26, 2019" %}

    {% blocktrans with link="https://www.mozillafestival.org/speakers/2019/algorithmic-gaze/" trimmed %} At MozFest 2019, Mozilla hosted “The Algorithmic Gaze,” a panel exploring the dangers of untrustworthy AI like YouTube’s recommendation engine. {% endblocktrans %}

  9. {% trans "December 25, 2019" %}

    {% blocktrans with link="https://foundation.mozilla.org/blog/congratulations-youtube-now-show-your-work/" trimmed %} Mozilla published a running list of YouTube’s dismissals related to its recommendation problem. {% endblocktrans %}

  10. {% trans "September 2020" %}

    {% trans "Mozilla launches the RegretsReporter browser extension to research YouTube’s recommendation problem." %}

  11. {% trans "July 2021" %}

    {% blocktrans with link="/campaigns/regrets-reporter/findings" trimmed %} Using data contributed by 37,380 RegretsReporter users, Mozilla releases first findings into YouTube's recommendations. {% endblocktrans %}
