{% trans "About Mozilla" %}

{% trans "Mozilla is fueling the movement for a healthy internet and trustworthy AI. We do that through projects like this, which hold companies accountable when their AI harms people. We’ve been pushing YouTube to address its recommendation problem for almost a year. Below, find a timeline of Mozilla’s work on this issue." %}

{% trans "A timeline of Mozilla’s work on this issue" %}

  1. {% blocktrans with date="July 2019" trimmed %}

    {{date}}

    Mozilla awards Guillaume Chaslot, a former YouTube engineer, a fellowship to support his work investigating AI systems, including the impacts of YouTube’s recommendation engine.

    {% endblocktrans %}
  2. {% blocktrans with date="August 7, 2019" link="https://foundation.mozilla.org/en/blog/youtubes-recommendations/" trimmed %}

    {{date}}

    Mozilla announced our intentions to investigate YouTube’s harmful recommendations, and to begin a dialogue with the company about making improvements. {% endblocktrans %}

  3. {% blocktrans with date="August 14, 2019" link="https://foundation.mozilla.org/en/blog/what-can-you-trust-internet/" trimmed %}

    {{date}}

    Mozilla published a reading list for our community detailing the ways recommendation AI — including YouTube’s — can manipulate consumers. {% endblocktrans %}

  4. {% blocktrans with date="September 3, 2019" link="https://foundation.mozilla.org/en/blog/answers-your-questions-about-dark-side-internet/" trimmed %}

    {{date}}

    Mozilla hosted a Q&A with a teenager who was radicalized online, partially due to YouTube’s recommendation AI.

    {% endblocktrans %}
  5. {% blocktrans with date="September 10, 2019" link="https://foundation.mozilla.org/en/blog/youtube-regrets/" trimmed %}

    {{date}}

    Mozilla launched its #YouTubeRegrets campaign, urging users to share stories about harmful recommendations they encounter.

    {% endblocktrans %}
  6. {% blocktrans with date="October 14, 2019" link="https://foundation.mozilla.org/en/blog/our-recommendation-youtube/" trimmed %}

    {{date}}

    After meeting with YouTube staff, Mozilla published three concrete actions the platform could take to help address its recommendation problem. These changes entail making YouTube’s recommendation data more transparent.

    {% endblocktrans %}
  7. {% blocktrans with date="October 15, 2019" link="https://foundation.mozilla.org/en/blog/28-reasons-why-youtube-must-change/" trimmed %}

    {{date}}

    Mozilla published findings from our #YouTubeRegrets research, highlighting how users are recommended anti-science content, anti-LGBT content, and other harmful videos.

    {% endblocktrans %}
  8. {% blocktrans with date="October 26, 2019" link="https://www.mozillafestival.org/en/speakers/2019/algorithmic-gaze/" trimmed %}

    {{date}}

    At MozFest 2020, Mozilla hosted “The Algorithmic Gaze,” a panel exploring the dangers of untrustworthy AI like YouTube’s recommendation engine.

    {% endblocktrans %}
  9. {% blocktrans with date="December 25, 2019" link="https://foundation.mozilla.org/en/blog/congratulations-youtube-now-show-your-work/" trimmed %}

    {{date}}

    Mozilla published a running list of YouTube’s deflections, denials, and misdirections related to its recommendation problem.

    {% endblocktrans %}
  10. {% blocktrans with date="September 2020" trimmed %}

    {{date}}

    Mozilla launches the RegretsReporter browser extension to more deeply interrogate YouTube’s recommendation problem.

    {% endblocktrans %}
